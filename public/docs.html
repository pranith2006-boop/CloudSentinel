Getting Started

CloudSentinel is an enterprise-grade cloud testing and observability platform. Follow these steps to begin:

# Clone the repository
$ git clone https://github.com/cloudsentinel/platform.git
# Install dependencies
$ npm install
# Start the server
$ node server.js

The server will start on port 4000 by default. Configure your MongoDB URI in the .env file.

API Endpoints
GET /api/metrics
Returns global dashboard metrics
POST /api/tests/trigger
Trigger a new test execution
GET /api/tests/recent
Fetch the 10 most recent test executions
POST /api/auth/login
Authenticate user credentials
Architecture Overview

Frontend

Vanilla HTML, CSS & JS with Chart.js for real-time data visualization and interactive particle effects.

Backend

Node.js + Express REST API with Mongoose ODM for MongoDB data persistence and test orchestration.

Database

MongoDB Atlas for storing test executions, metrics snapshots, and historical performance data.
